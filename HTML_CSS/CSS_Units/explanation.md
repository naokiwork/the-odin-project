<ul> とは？

✔ 役割

丸ポチ（●）の箇条書きリストを作るタグ。

“unordered list”（順番のないリスト）の略。

✔ イメージ

ノートに「今日やることリスト」を書く時の ・（黒丸） と同じ。
・洗濯
・勉強
・ご飯

<ul>
  <li>洗濯</li>
  <li>勉強</li>
  <li>ご飯</li>
</ul>











 <li> とは？

✔ 役割

リストの項目1つ1つを表すタグ。

“list item” の略。

✔ ルール
	•	li は必ず ul（または ol）の 子供 になる
	•	単体では使えない








class="ems" は何？

<ul class="ems">
これは “ems” という名前の CSS を当てたい という意味。
.ems li {
  font-size: 1.5em;
  color: blue;
}
