vh
あなたがスマホ画面を持っていて、画面の“高さ”を100として考える。
	•	100vh = 画面の高さそのまま
	•	50vh = 画面の高さの半分
	•	1vh = 画面の高さの1%



vw = 画面の幅を100として考える単位
画面の“横幅”を100として扱う。
	•	100vw = 画面の横幅ぜんぶ
	•	10vw = 横幅の10%
	•	1vw = 横幅の1%
