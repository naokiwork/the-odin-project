✅ アクティブリコール問題（答え＋解説編）

A1. Background

A1-1.

答え
👉 8個

ファインマン解説
background は「背景に関する全部入りセット」。
色・画像・位置・サイズ・繰り返しなどを一気に指定できる便利パック。

⸻

A1-2.

答え（例）
	•	背景画像（background-image）
	•	背景画像の位置（background-position）
	•	背景画像のサイズ（background-size）
	•	繰り返し方法（background-repeat）

ファインマン解説
👉 背景は「色」だけでなく「敷き詰める絵」も扱える。

⸻

A1-3.

答え
👉 可読性や明確さを重視したいとき

ファインマン解説
background は便利だが情報量が多い。
「何を設定しているか」を明確にしたい場合は
background-color などを個別に書いた方が安全。

⸻

A2. Borders

A2-1.

答え
👉 太さ / 線の種類 / 色

例：

border: 1px solid black;

ファインマン解説
👉 枠線は「どれくらい太いか」「どんな線か」「何色か」の3点で決まる。

⸻

A2-2.

答え
👉 角を丸くする

ファインマン解説
👉 四角い箱の角を削るイメージ。

⸻

A2-3.

答え
👉 必要になるケースがほとんどないから

ファインマン解説
実務では「全部同じ丸さ」で十分なことが多い。
細かい指定は「必要になったら調べる知識」。

⸻

A3. Box-shadow

A3-1.

答え
👉 奥行き・立体感・要素の分離を表現するため

ファインマン解説
👉 影があると「浮いて見える」＝重要に見える。

⸻

A3-2.

答え
👉 強すぎる影はデザインをうるさくするから

ファインマン解説
良い影は「気づかれない影」。
存在感はあるが主張しない。

⸻

A4. Overflow

A4-1.

答え
👉 中身がはみ出たときの挙動を決める

ファインマン解説
👉 箱に入りきらないものを「どう扱うか」のルール。

⸻

A4-2.

答え
👉 スクロール可能なエリアを作ること

ファインマン解説
カードUIやチャット欄などでよく使われる。

⸻

A5. Opacity

A5-1.

答え
👉 要素の透明度

ファインマン解説
0 = 完全透明
1 = 完全不透明

⸻

A5-2.

答え（例）
	•	ホバー時の演出
	•	要素の重なり表現

ファインマン解説
👉 「今選ばれている」「背後にある」を直感的に伝えられる。
